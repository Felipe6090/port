Olá, esse é um portifolio feito usando nextJS

## Getting Started

Comandos para rodar o server de desenvolvimento:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.
